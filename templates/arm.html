This is an unsupported variant, as x86 is the only CPU architecture being targeted for R1. It is provided for development and testing purposes only. Releasing software that requires these unsupported builds is discouraged.

The ARM port is in an extremely early state. Ongoing work to convert Haiku over to an ARM EFI bootloader means these likely won't function. (yet)
These generic ARM images target qemu with an EFI bootloader. They can also be processed by an installation tool to run on a wide range of small ARM devices including the Raspberry Pi (2+), Nano Pi, OrangePi, Tinkerboard, and more.
