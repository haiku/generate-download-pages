<%inherit file="base.html"/> <%block name="title">x86_64 - Unsupported Builds <%block name="description">

x86_64 is an unsupported variant, as it lacks the ability to run GCC 2 binaries and on 32-bit Intel compatible CPUs. It is provided for development and testing purposes only. Releasing software that requires these unsupported builds is discouraged.
